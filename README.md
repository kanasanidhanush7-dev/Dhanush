# Linux Administration Homelab Project

## Project Overview

This project demonstrates hands-on Linux Administration skills using Ubuntu in VirtualBox. The environment was built to simulate real-world system administration tasks.

## Technologies Used

* Ubuntu Linux
* VirtualBox
* OpenSSH
* NFS
* Samba
* LVM
* Git & GitHub

## Tasks Performed

* User and Group Management
* File Permissions and ACL Configuration
* SSH Server Setup
* Logical Volume Management (LVM)
* NFS Server Configuration
* Samba File Sharing
* System Monitoring and Troubleshooting
* Git Version Control
